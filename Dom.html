DOM Manipulation Example.


Neste exemplo, o código JavaScript espera que o conteúdo do DOM seja totalmente carregado
( DOMContentLoadedevento) antes de ser executado.
Em seguida, ele cria um novo pelemento (parágrafo), define seu conteúdo de texto,
adiciona um atributo de classe e, finalmente, anexa-o ao corpo do documento HTML.
O parágrafo será exibido na página da web quando for carregado.
